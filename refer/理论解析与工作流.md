## 1. 理论基础

### 1.1 光锥 PDF 的定义

质子中非极化胶子的光锥 PDF $g(x, \mu^2)$ 定义为光锥关联函数的 Fourier 变换：

$$g(x, \mu^2) = \int \frac{d\xi^-}{2\pi x P^+} e^{-ixP^+\xi^-} \langle P | F^{+\mu}(\xi^-) W(\xi^-, 0) \tilde{F}_\mu^{+}(0) | P \rangle$$

其中：
- $P^+ = (P^0 + P^3) / \sqrt{2}$ 是光锥坐标下的正向动量
- $F^{\mu\nu}$ 是胶子场强张量
- $\tilde{F}^{\mu\nu} = \frac{1}{2}\varepsilon^{\mu\nu\rho\sigma}F_{\rho\sigma}$ 是对偶场强张量
- $W(\xi^-, 0)$ 是连接 $\xi^-$ 和 $0$ 的光锥 Wilson 线
- $x$ 是 Bjorken-$x$（胶子携带的纵向动量分数）
- $\mu$ 是重整化标度

### 1.2 大动量有效理论 (LaMET)

LaMET (Large-Momentum Effective Theory) 的核心思想是：

> 在 Euclidean 格点上计算一个快速移动的强子的**空间方向**关联函数（称为 quasi-PDF），然后在大的强子动量 $P_z$ 下，通过微扰匹配将其与光锥 PDF 联系起来。

关键点：
- **Quasi-PDF** $\tilde{q}(x, P_z)$：对空间分离的关联函数做 Fourier 变换得到
- **大动量极限** $P_z \to \infty$：quasi-PDF 趋近于光锥 PDF
- **有限动量** $P_z$：通过**因子化定理**，quasi-PDF 和光锥 PDF 之间存在可微扰计算的匹配关系

LaMET 的有效性依赖于因子化：

$$\tilde{q}(x, P_z) = \int_{0}^{1} \frac{dy}{|y|} C\left(\frac{x}{y}, \frac{\mu}{P_z}\right) g(y, \mu) + \mathcal{O}\left(\frac{\Lambda_{\text{QCD}}^2}{P_z^2}, \frac{\Lambda_{\text{QCD}}^2}{x^2 P_z^2}\right)$$

其中 $C$ 是**匹配核** (matching kernel)，可以通过微扰 QCD 计算。

### 1.3 准 PDF 与匹配关系

对于胶子 quasi-PDF，空间方向（取 $z$-方向为 boost 方向）的 quasi-PDF 定义为：

$$\tilde{g}(x, P_z) = \int \frac{dz}{2\pi x P_z} e^{ixP_z z} \tilde{h}(z, P_z)$$

其中**坐标空间矩阵元** $\tilde{h}(z, P_z)$ 为：

$$\tilde{h}(z, P_z) = \frac{\langle P | O(z) | P \rangle}{\langle P | P \rangle}$$

这就是 note 中的 **Eq(20)** 的核心内容。

quasi-PDF 到光锥 PDF 的匹配关系为 ($\mu$ 为 $\overline{\text{MS}}$ 标度)：

$$\tilde{g}(x, P_z) = g(x, \mu) + \frac{\alpha_s}{2\pi} \int_x^1 \frac{dy}{y} \Delta C_{gg}\left(\frac{x}{y}, \frac{\mu}{P_z}\right) g(y, \mu) + \cdots$$

### 1.4 胶子 quasi-PDF 算符

胶子 unpolarized quasi-PDF 算符在空间方向上定义为：

$$O_{\mu\nu}(z) \equiv F_{z\mu}(z\hat{n}_z) \, W(z\hat{n}_z, 0) \, \tilde{F}_{\nu}^{\;z}(0)$$

其中 $\hat{n}_z$ 是空间 $z$ 方向的单位矢量。

对 unpolarized 情形，我们需要的指标收缩为：

$$O(z) = O_{\mu}^{\;\mu}(z) = F_{z\mu}(z) \, W(z, 0) \, \tilde{F}^{\mu z}(0)$$

**Eq(25)** 给出了我们具体拆解出的 OPE 算符形式（用 $F$ 和 $\tilde{F}$ 的不同 Lorentz 分量组合）。

对于格点计算，由于旋转对称性破缺，我们需要对等价的方向进行平均。在 $z$-方向 boost 的情况下，横向指标 $\mu, \nu \in \{x, y, t\}$ 需要求和：

$$O_{\text{unpol}}(z) \equiv \sum_{\mu \neq z} \left[ F_{z\mu}(z) W(z, 0) \tilde{F}^{\mu z}(0) + F_{z\mu}(z) W(z, 0) \tilde{F}^{\mu z}(0) \right]_{\text{对称化}}$$

## 2. 格点离散化

### 2.1 场强张量的格点构造 (Clover 定义)

在格点上，胶子场强张量 $F_{\mu\nu}$ 不能直接定义，需要通过规范链接 $U_\mu(x)$ 构造。

**Plaquette（方格）** 在 $\mu$-$\nu$ 平面上的定义为：

$$P_{\mu\nu}(x) = U_\mu(x) U_\nu(x+\hat{\mu}) U_\mu^\dagger(x+\hat{\nu}) U_\nu^\dagger(x)$$

**Clover 算符**：通过对围绕点 $x$ 的四个 plaquette 取反对称化组合，得到 $\mathcal{O}(a^2)$ 改进的场强张量：

$$\hat{F}_{\mu\nu}(x) = -\frac{i}{8a^2 g_0} \left( Q_{\mu\nu}(x) - Q_{\mu\nu}^\dagger(x) \right)$$

其中 $Q_{\mu\nu}$ 是 "clover leaf"——四个 plaquette 的和：

$$Q_{\mu\nu}(x) = P_{\mu\nu}(x) + P_{\nu,-\mu}(x) + P_{-\mu,-\nu}(x) + P_{-\nu,\mu}(x)$$

四个 plaquette 的几何含义：

```
        μ
        ↑
        |  P(ν,-μ)    P(μ,ν)
        |    (2)        (1)
        +----x----→----+
        |              |
        |    (3)   (4)  |
        +----←----x    |
        P(-μ,-ν)  P(-ν,μ)
                      ν →
```

具体构造（对应 `plaquette_clover_new` 函数）：

```
P_{μν}(x):      x → x+μ → x+μ+ν → x+ν → x
P_{ν,-μ}(x):    x → x+ν → x+ν-μ → x-μ → x
P_{-μ,-ν}(x):   x → x-μ → x-μ-ν → x-ν → x
P_{-ν,μ}(x):    x → x-ν → x-ν+μ → x+μ → x
```

最终 clover 场强张量：

$$\hat{F}_{\mu\nu}(x) = -\frac{i}{8} \left[ P_{\mu\nu} + P_{\nu,-\mu} + P_{-\mu,-\nu} + P_{-\nu,\mu} - \text{h.c.} \right]$$

在代码中，`plaquette_clover_new` 函数计算了这四个 plaquette 及其反厄米部分的组合。

### 2.2 对偶场强张量

对偶场强张量 $\tilde{F}_{\mu\nu}$ 通过 Levi-Civita 张量与 $F$ 联系：

$$\tilde{F}_{\mu\nu} = \frac{1}{2} \varepsilon_{\mu\nu\rho\sigma} F^{\rho\sigma}$$

其中 $\varepsilon_{\mu\nu\rho\sigma}$ 是四维完全反对称张量，约定 $\varepsilon_{0123} = +1$。

在代码中，`Tensor4` 数组实现了 $\frac{1}{2}\varepsilon_{\mu\nu\rho\sigma}$：

$$
\varepsilon_{\mu\nu\rho\sigma} = \begin{cases}
+1 & \text{如果 } (\mu,\nu,\rho,\sigma) \text{ 是 } (0,1,2,3) \text{ 的偶排列} \\
-1 & \text{如果是奇排列} \\
0 & \text{如果有重复指标}
\end{cases}
$$

`plaquette_clover_all_tilde` 函数通过缩并实现 $\tilde{F}$：

```python
pla_tilde[μ, ν] = ε_{μνρσ} * pla[ρ, σ]
```

### 2.3 Wilson 线

连接 $0$ 和 $z\hat{n}$ 的 Wilson 线在格点上是一个路径有序乘积：

$$W(z\hat{n}, 0) = \mathcal{P} \exp\left( i g_0 \int_0^z dz' A_z(z'\hat{n}) \right)$$

在格点上离散化为规范链接的乘积：

$$W(z\hat{n}_z, 0) \approx \prod_{k=0}^{z/a-1} U_z(k a \hat{n}_z)$$

代码实现（`operators_new_z0` 中的 Wilson 线构造）：

```python
# 从 z 到 0 的 Wilson 线 (取共轭 = 反向)
for _dz in range(delta_z):
    ope = np.einsum("tzyxab, tzyxcb -> tzyxac",
                     ope,
                     gauge[:,:,:,:,z_dir,:,:].conj()[rolled])
# 从 0 到 z 的 Wilson 线
for _dz in range(delta_z):
    ope = np.einsum("tzyxab, tzyxbc -> tzyxac",
                     ope,
                     gauge[:,:,:,:,z_dir,:,:][rolled])
```

### 2.4 完整胶子算符的格点实现

完整的 nonlocal 胶子算符 $O_{\mu\nu}(z)$ 的格点表示为：

$$O_{\mu\nu}^{\text{lat}}(z) = \hat{F}_{z\mu}(z\hat{n}_z) \left[ \prod_{k=0}^{z/a-1} U_z(k a) \right] \hat{\tilde{F}}_{\nu}^{\;z}(0)$$

在代码中，这对应于 `operators_new_z0_mu2` 函数：

1. 将 $\hat{F}_{\mu\nu}$ (plaquette) 平移到 z 位置
2. 从 z 到 0 连接 Wilson 线（U_z 的共轭）
3. 在 0 处插入 $\hat{\tilde{F}}_{\mu2,\nu2}$
4. 从 0 到 z 连接 Wilson 线（U_z）
5. 对颜色空间取迹（trace）

$$\text{Tr}_c\left[ \hat{F}_{\mu\nu}(z) \cdot W(z \to 0) \cdot \hat{\tilde{F}}_{\mu2,\nu2}(0) \cdot W(0 \to z) \right]$$

## 3. 关联函数分解

### 3.1 三点关联函数与 disconnect 图

我们需要计算的三点关联函数为：

$$C_3(\Delta t; z, P_z) = \langle \Omega | \mathcal{N}(\mathbf{P}, t_{\text{sink}}) \, O(z; t_{\text{op}}) \, \bar{\mathcal{N}}(\mathbf{P}, t_{\text{src}}) | \Omega \rangle$$

其中 $\mathcal{N}$ 是质子插值算符（interpolating operator），$O(z)$ 是胶子 quasi-PDF 算符。

对于胶子算符，只有 **disconnected 图**（也称为 "quark-disconnected" 图）有贡献：

```
   N(t_sink) ──── ──── N̄(t_src)
                     │
                     O(z; t_op)
```

**关键近似**：三点关联函数解耦为质子两点函数和 OPE 算符的乘积：

$$C_3(\Delta t; z, P_z) \approx C_2(\Delta t; P_z) \cdot \langle O(z) \rangle_{\text{gauge}}$$

其中：
- $C_2(\Delta t; P_z) = \langle \mathcal{N}(t_{\text{sink}}) \bar{\mathcal{N}}(t_{\text{src}}) \rangle$ 是质子两点关联函数
- $\langle O(z) \rangle_{\text{gauge}}$ 是纯规范场中 OPE 算符的真空期望值

这个分解基于：
1. Disconnected 图中胶子算符与夸克线解耦
2. 夸克部分构成标准的质子两点函数
3. 胶子部分可以在纯规范组态上独立计算

### 3.2 Eq(20): 矩阵元

**Eq(20)** 给出了我们要计算的矩阵元：

$$h(z, P_z) \equiv \frac{\langle P | O(z) | P \rangle}{\langle P | P \rangle}$$

在格点上通过关联函数比提取：

$$h^{\text{lat}}(z, P_z; t_{\text{sep}}) = \frac{C_3(t_{\text{sep}}; z, P_z)}{C_2(t_{\text{sep}}; P_z)} \xrightarrow{t_{\text{sep}} \to \infty} h(z, P_z)$$

在大时间分离极限下，激发态污染被压制，比值逼近基态矩阵元。

利用上述解耦：

$$h^{\text{lat}}(z, P_z) \approx \langle O(z) \rangle_{\text{gauge}}$$

即纯规范场中 OPE 算符的期望值（对所有规范组态平均）。

### 3.3 Eq(25): OPE 算符

**Eq(25)** 给出了我们拆分出的 OPE 算符的具体形式。

对于非极化胶子，算符的 Lorentz 结构为：

$$O_{\text{unpol}}(z) = F_{z\mu}(z) W(z, 0) \tilde{F}^{\mu z}(0)$$

在格点上，我们需要对 $\mu \neq z$ 的所有方向求和（且考虑 $F$ 和 $\tilde{F}$ 的独立 Lorentz 指标）：

$$O(z) = \sum_{\mu \neq z} \sum_{\nu \neq z} g^{\mu\nu} \, F_{z\mu}(z) \, W(z, 0) \, \tilde{F}_{\nu}^{\;z}(0)$$

在实际计算中（对应 `Calc_ope_unpol.py` 中的 MPI 并行方案）：

- rank 0: $(\mu,\nu) = (t, x)$ 当 $z = y$
- rank 1: $(\mu,\nu) = (t, y)$ 当 $z = x$
- rank 2: $(\mu,\nu) = (x, y)$ 当 $z = t$

不同 Lorentz 分量的计算分配给不同的 MPI 进程，实现并行化。

## 4. Distillation 方法与动量涂抹

### 4.1 Distillation 的基本思想

**Distillation** (Peardon et al., 2009) 是一种夸克场涂抹方法，用于构建高质量的强子插值算符。

基本思想：
1. 计算格点 Laplace 算符的低模本征矢量 $v^{(k)}(x)$
2. 用截断的本征矢量展开构建**perpambulator**（夸克传播子的子空间投影）
3. 所有 Wick 收缩在 $N_{\text{ev}} \times N_{\text{ev}}$ 的子空间内完成

夸克传播子的 Distillation 表示：

$$S(x, y)_{\alpha\beta}^{ab} \approx \sum_{k,l=1}^{N_{\text{ev}}} v^{(k)}(x) \, \tau^{(kl)}(t) \, v^{(l)\dagger}(y)$$

其中 $\tau$ 是 perambulator，在 $N_{\text{ev}} \times N_{\text{ev}}$ 的子空间内。

**优势**：
- 计算复杂的多夸克关联函数变得可行
- 本征矢量可以在不同算符之间复用
- 自然实现了夸克场的涂抹，压制了 UV 涨落

### 4.2 动量涂抹 (Momentum Smearing)

为了在大动量 $P_z$ 下有效计算质子关联函数，我们使用**动量涂抹**技术 (Bali et al., 2016)：

$$\tilde{v}^{(k)}(x; \mathbf{k}) = e^{i \mathbf{k} \cdot \mathbf{x}} \, v^{(k)}(x)$$

其中 $\mathbf{k}$ 是涂抹动量，与物理动量 $P_z$ 匹配。

在代码中（`phase_calc` 函数）：

```python
phase_factor[x] = exp(-i * Mom · x * 2π / Nx)
eigvecs_mom2 = contract("vxa, x -> vxa", eigvecs, phase_factor)
```

动量涂抹后的本征矢量用于计算包含动量投影的 VVV（baryon block）。

### 4.3 质子两点关联函数的 Distillation 构造

质子插值算符（以 $C\gamma_5$ 为例）：

$$\mathcal{N}_\alpha(x) = \varepsilon^{abc} \left[ u^a(x)^T C\gamma_5 d^b(x) \right] u^c_\alpha(x)$$

在 Distillation 框架下，质子两点关联函数的收缩为：

$$C_2(\Delta t; \mathbf{P}) = \sum_{t_{\text{src}}} \sum_{\mathbf{x}, \mathbf{y}} e^{-i\mathbf{P}\cdot(\mathbf{y}-\mathbf{x})} \langle \mathcal{N}(\mathbf{y}, t_{\text{sink}}) \bar{\mathcal{N}}(\mathbf{x}, t_{\text{src}}) \rangle$$

收缩需要以下元素 (代码实现)：
1. **VVV (Baryon Block)**：三夸克本征矢量的动量投影组合
2. **Perambulator** $\tau(t_{\text{sink}}, t_{\text{src}})$：夸克传播子的子空间表示
3. **Dirac 矩阵插入**：$C\gamma_5$ 或 $C\gamma_5\gamma_\mu$ 插值算符

核心收缩公式（代码中的 `contract` 调用）：

$$\begin{aligned}
C_2^{ij}(t_{\text{sink}}, t_{\text{src}}) = &\;\Phi_{abc} \, \tau^{ad}(t_{\text{sink}}, t_{\text{src}})_{i\alpha} \, [\Gamma \tau(t_{\text{sink}}, t_{\text{src}}) \Gamma]^{be}_{\beta\gamma} \, \tau^{cf}(t_{\text{sink}}, t_{\text{src}})_{j\delta} \, \Phi_{def}^* \\
& - \Phi_{abc} \, \tau^{af}(t_{\text{sink}}, t_{\text{src}})_{i\alpha} \, [\Gamma \tau(t_{\text{sink}}, t_{\text{src}}) \Gamma]^{be}_{\beta\gamma} \, \tau^{cd}(t_{\text{sink}}, t_{\text{src}})_{j\delta} \, \Phi_{def}^*
\end{aligned}$$

其中 $\Phi_{abc}$ 是 VVV 张量（baryon block），包含三个本征矢量的色收缩和动量相位因子。

| `eigvecs` | Laplace 本征矢量 $v^{(k)}(x)$ | $(N_{\text{ev}}, N_x N_y N_z, 3)$ |
| `peram` | Perambulator $\tau(t_{\text{sink}}, t_{\text{src}})$ | $(N_t, 4, 4, N_{\text{ev}}, N_{\text{ev}})$ |
| `VVV` | Baryon block $\Phi_{abc}$ | $(N_{\text{ev}}, N_{\text{ev}}, N_{\text{ev}})$ |

**参数**：
- 格点大小：$L^3 \times T = 32^3 \times 64$（典型值）
- 本征矢量数：$N_{\text{ev}} = 100$

- 动量涂抹：$\mathbf{k} = (0, 0, -3)$ 匹配 boost 方向
- Wilson 线长度：$z_{\text{max}} \sim 10\text{-}15$ 格距

### 5.2 OPE 计算

**目标**：计算 $\langle O(z) \rangle$，即在纯规范组态上对 OPE 算符求平均。

**算法流程**：

```
Algorithm: OPE 计算
────────────────────────────────────────────────────
Input:  gauge links U_μ(x), 格点尺寸 (Nt, Nx), delta_z
Output: OPE(z) for z = 0, ..., delta_z-1

1. for each (μ, ν) pair (μ ≠ ν):           ▸ 并行 (MPI)
2.     pla[μ, ν] = plaquette_clover_new(gauge, μ, ν)
         → F_{μν}(x), size: (Nt, Nx, Nx, Nx, 3, 3)
3. pla_tilde[μ, ν] = ε_{μνρσ} · pla[ρ, σ]
         → F̃_{μν}(x)
4. for dz in 0 ... delta_z-1:
5.     平移 F_{μν} 到 z = dz
6.     从 z 到 0: 插入 U_z^† (Wilson 线共轭)
7.     在 0 处: 插入 F̃_{μ2, ν2}
8.     从 0 到 z: 插入 U_z (Wilson 线)
9.     对颜色空间取 trace: Tr_c[...]
10.    对空间求和: Σ_{x,y}
11.    ops[t, dz] ← 上述结果
12. end for
13. 保存结果: ops_mu{μ}_nu{ν}_dz{dz}_conf{conf_id}.npy
```

**并行策略**：不同的 Lorentz 指标组合 $(\mu, \nu, \mu_2, \nu_2)$ 分配给不同的 MPI rank：

| Rank | $(\mu, \nu)$ | $(\mu_2, \nu_2)$ | 物理含义 |
|------|-------------|-----------------|---------|
| 0 | $(t, x)$ 或 $(t, y)$ | $(t, x)$ 或 $(t, y)$ | $F_{zt} \tilde{F}^{tz}$ |
| 1 | $(t, y)$ 或 $(t, z)$ | $(t, y)$ 或 $(t, z)$ | $F_{zt} \tilde{F}^{tz}$ (另一分量) |
| 2 | $(x, y)$ | $(x, y)$ | $F_{zx} \tilde{F}^{xz}$ 或 $F_{zy} \tilde{F}^{yz}$ |

**注意**：$\mu, \nu$ 的具体选择取决于 Wilson 线方向 $z_{\text{dir}}$：
- $z_{\text{dir}} = 0 (x)$: $\nu = 1 (y)$ 或 $\nu = 2 (z)$
- $z_{\text{dir}} = 1 (y)$: $\nu = 0 (x)$ 或 $\nu = 2 (z)$
- $z_{\text{dir}} = 2 (z)$: $\nu = 0 (x)$ 或 $\nu = 1 (y)$

### 5.3 质子 2pt 计算

**目标**：计算动量投影的质子两点关联函数 $C_2(\Delta t, \mathbf{P})$。

**GPU 加速策略**：
- 使用 CuPy 进行 GPU 上的张量缩并
- VVV 的动量投影在 GPU 上逐层分片计算（显存限制）
- Perambulator 在 GPU 上每个 $t_{\text{src}}$ 分别加载

**算法流程**：

```
Algorithm: 质子 2pt (Distillation + Momentum Smearing)
────────────────────────────────────────────────────────
Input:  eigvecs, peram, VVV, 动量 P = (Px, Py, Pz)
Output: C₂(t_sink, t_src) for Dirac indices

1. for each Px in [3, 4, 5, 6, 7, 8]:
2.     VVV_sink[t] = Σ_x e^{-iP·x} ε_{abc} v^a(x) v^b(x) v^c(x)
         ▸ GPU: 逐 x-层计算，累加到 VVV_sink[t]
3.     for each t_src:
4.         加载 perambulator τ(t_sink, t_src)
5.         插入 Dirac 矩阵: Γ = Cγ₅γ_μ
6.         CG5_tau_CG5 = Γ · τ · Γ
7.         for each t_sink:
8.             收缩 (直接项):
                 C₂₁ = Φ(t_sink) · τ(t_sink) · CG5τ · τ(t_sink) · Φ(t_src)*
9.             收缩 (交换项):
                 C₂₂ = Φ(t_sink) · τ_swap · CG5τ · τ_swap · Φ(t_src)*
10.            C₂(t_sink, t_src) = C₂₁ - C₂₂
11.        end for
12.    end for
13.    宇称投影:
         C₂⁺ = P₊ · C₂  (正宇称)
         C₂⁻ = P₋ · C₂  (负宇称)
14.    边界条件符号修正
15.    保存结果
16. end for
```

**VVV 收缩细节**（`VVV_Calc_cupy` 函数）：

$$\Phi_{abc}(t, \mathbf{P}) = \sum_{\mathbf{x}} e^{-i\mathbf{P} \cdot \mathbf{x}} \varepsilon_{ijk} \, v^{(a)}_i(\mathbf{x}, t) \, v^{(b)}_j(\mathbf{x}, t) \, v^{(c)}_k(\mathbf{x}, t)$$

这里 $i, j, k \in \{0, 1, 2\}$ 是色指标，$a, b, c$ 是本征矢量指标。

### 5.4 矩阵元提取

从关联函数中提取胶子 quasi-PDF 矩阵元：

**步骤**：

1. **计算有效矩阵元**：

   $$h^{\text{eff}}(z, P_z; \Delta t) = \frac{C_3(\Delta t; z, P_z)}{C_2(\Delta t; P_z)}$$

   在 disconnect 近似下：

   $$h^{\text{eff}}(z, P_z; \Delta t) \approx \langle O(z) \rangle_{\Delta t}$$

2. **Plateau 拟合**：选择 $\Delta t$ 足够大的区域，使得激发态污染可以忽略：

   $$h(z, P_z) = \lim_{\Delta t \to \infty} h^{\text{eff}}(z, P_z; \Delta t) \approx \text{const}$$

3. **统计平均**：

   $$h(z, P_z) = \frac{1}{N_{\text{conf}}} \sum_{\text{conf}} \langle O(z) \rangle_{\text{conf}}$$

4. **误差估计**：使用 Jackknife 或 Bootstrap 方法

### 5.5 Fourier 变换与匹配

**坐标空间 → 动量空间**：

准 PDF 通过 Fourier 变换得到：

$$\tilde{g}(x, P_z) = \frac{2P_z}{x} \int_0^{z_{\text{max}}} dz \, h_R(z, P_z) \sin(x P_z z)$$

其中 $h_R(z, P_z)$ 是重整化后的矩阵元。

**重整化**：

在混合方案中：
- 短距离 ($z < z_s$)：使用比值重整化 $h_R(z) = h(z) / h(z=0)$
- 长距离 ($z > z_s$)：不做重整化（或使用替代方案）

实际上，对于胶子，常采用**自重整化** (self-renormalization) 方案，利用 $z=0$ 处的矩阵元作为重整化因子。

**匹配到光锥 PDF** (NER = 微扰匹配核)：

$$g(x, \mu) = \tilde{g}(x, P_z) - \frac{\alpha_s}{2\pi} \int_x^1 \frac{dy}{y} \Delta C_{gg}\left(\frac{x}{y}, \frac{\mu}{P_z}\right) \tilde{g}(y, P_z)$$

匹配核 $\Delta C_{gg}$ 可以微扰计算到 NLO。
